Mathematical Basis of ML

Posted on 2017-09-30 | In ML | Views

This article is divided into three parts, namely linear algebra, probability theory, differential learning

1. Linear Algebra Review

Matrix multiplication

  • (AB)C = A(BC)
  • A(B + C) = AB + AC
  • [Note] : AB != BA

Transpose Matix

  • $(A^T)^T = A$
  • $(AB)^T = B^TA^T$
  • $(A + B)^T = A^T + B^T$

[Note] : $A \in \mathbb{R} $ is symmetric if $A^T = A$

Trace

  • $trA = trA^T$
  • $tr(A + B) = trA + trB$
  • $tr(tA) = t trA$
  • $trAB = trBA$
  • $trABC = trBCA = trCAB$

[Note] : $A, B, C \in \mathbb{R^2}$

Inverse

  • $(A^{-1})^{-1} = A$
  • $(AB)^{-1} = B^{-1}A^{-1}$
  • $(A^{-1})^T = (A^T)^{-1} = A^{-T}$

Determinant

  • $|A| = |A^T|$
  • $|AB| = |A||B|$
  • $|A|^{-1} = \frac{1}{|A|}$

2. Probability Theory

Random Variables

Random variables play an important role in probability theory. The most important fact about random variables is that they are not variables. They are actually functions that map outcomes (in the outcome space) to real values. In terms of notation, we usually denote random variables by a capital letter.
