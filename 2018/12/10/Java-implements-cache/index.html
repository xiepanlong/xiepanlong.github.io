Java实现缓存机制

基于三种策略（FIFO/LRU/LFU），通过java实现简单的缓存机制，并且通过可重入锁保证其线程安全。
(1)FIFO——先进先出
(2)LRU——最近最少使用：最近读取的数据放在最前面，最早读取的数据放在最后面
(3)LFU——最不常使用：对存储的数据都会有一个计数引用，然后队列按数据引用次数排序，引用数多的排在最前面，引用数少的排在后面。
