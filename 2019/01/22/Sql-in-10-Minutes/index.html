1 SQL的基本概念

SQL——Structured Quert Language（结构化查询语言）的缩写。SQL是专门用来和数据库沟通的语言。

2 检索

  1. DISTINCT
    用于检索数据中返回不同的值，值得注意的是：DISTINCT关键字作用于所有的列，而不是仅仅跟在其后的那一列。而且写法上DISTINCT必须写在所有列的前面，不然会报错。

  2. 限制结果
    MySql中使用Limit来返回第一行或者一定数量的行。例如：

    SELECT FROM tableName LIMIT 5; — 返回查询结果的前五条
    SELECT     FROM tableName LIMIT 3, 4; — LIMIT OFFSET, ROWS 从偏移量offset开始的rows行

  3. 排序检索结果
    使用ORDER BY子句进行对查询结果排序。例如：

    1. SELECT * FROM tableName ORDER BY colName;
      — 按照colName对查询结果进行排序，默认降序；注意的地方有ORDER BY 子句的位置必须在查询语句的末尾，与此同时也可以使用非选择列进行排序。
    2. SELECT * FROM tableName ORDER BY colName1 DESC, colName2 ASC;
      — 按照colName1降序，colName2升序对查询结果进行排序；

  4. 过滤数据
    使用SELECT和WHERE子句组合来指定检索数据。

注释：
